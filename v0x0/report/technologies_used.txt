Technologies Used\nThe stack, evident in the files, balances performance and accessibility:  \nNext.js: Server-side rendering and API routes.  \n\nTypeScript: Type safety with Zod schemas (lib/schemas.ts).  \n\nOpenAI (GPT-4o-mini): AI image analysis (lib/actions.ts).  \n\nPrisma: Database ORM (lib/prisma.ts).  \n\nSupabase: Image storage (lib/supabase.ts).  \n\nNextAuth.js: Authentication (auth.ts).  \n\nGeolocation APIs: ipgeolocation.io and bigdatacloud.net (app/api/geolocation/route.ts).\nReal-World Example: MobilePlantViT uses AI for plant diagnostics (arXiv).
